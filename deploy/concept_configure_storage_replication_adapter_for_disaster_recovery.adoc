= 設定儲存複寫介面卡以進行災難恢復
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您要設定vCenter Server進行災難恢復、則必須在部署之後啟用儲存複寫介面卡（SRA）。根據預設、ONTAP 部署這些功能會安裝VSC。部署ONTAP 完各種VMware工具之後、您必須為vCenter Server啟用SRA。
